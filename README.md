# XzPlayer - 播放器

轻量 PHP 在线视频解析播放器，基于 ArtPlayer 开发，支持 M3U8 播放、广告过滤，跨端适配、部署即用。

## ✨ 核心功能

- 播放格式：支持 M3U8 视频格式在线解析与播放

- 广告体验：智能过滤视频广告，支持自定义广告屏蔽关键词

- 终端体验：电脑、手机全终端适配，观影流畅无卡顿

## 📋 环境需求

- PHP 7.4+（推荐 8.0+），需开启 cURL 扩展

- 支持 Nginx/Apache/IIS 等主流 Web 服务器

- 浏览器：Chrome 60+/Firefox 55+/Safari 12+/Edge 79+

- 项目目录需具备读写权限（用于缓存与会话存储）
